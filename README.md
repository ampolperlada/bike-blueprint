## Features

- 🏍️ **3D Customizer** — Rotate, customize, and preview an NMAX 155 in real-time
- 🎨 **22+ Colors** — Yamaha, Honda, and custom paint presets
- 🛒 **Parts Catalog** — Real brands (RCB, Rizoma, Akrapovic) with pricing in ₱
- 📊 **Performance Stats** — See acceleration, handling, and weight impact per part
- 💾 **Save & Share** — Save up to 10 builds, share via URL, export as PNG
- 📍 **Shop Locator** — Connect with certified installers via WhatsApp/Messenger

## Tech Stack

**Next.js 16** · **TypeScript 5** · **Three.js r160** · **Tailwind CSS** · **Docker**

---

## Quick Start
```bash
git clone https://github.com/ampolperlada/bike-blueprint
cd 3d-moto-sys
npm install
npm run dev
# → http://localhost:3000
```

Or with Docker:
```bash
docker-compose up
```
